INTRODUCTION

Gather Coffee is a website designed to make you feel like you are drinking coffee face-to-face with friends and family. Web simulates the model of coffee shops to help users have the most authentic atmosphere, besides the website also has the same functions as when we go to drink coffee at the shop such as ordering drinks, paying, ordering table and chat with friends via chat or video call. The design team firmly believes that with the above functions and the coffee shop model, it will create excitement, novelty and great feeling for users.

ADDITION

Besides creating a feeling like when we go to a coffee shop with friends and relatives at the shop. The group also wants to use this website so that users can connect with each other to help expand their personal relationships. Users can connect with each other through preferences about drinks, seats, favorite shops, etc., thereby making it easier for people to start conversations and make friends.
